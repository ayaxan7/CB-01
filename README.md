# CB-01 Cyberpunk Badge


### A Cyberpunk-Inspired RP2040 Electronic Badge

A custom-designed cyberpunk-themed electronic badge built around the RP2040 microcontroller, featuring RGB lighting, OLED display support, onboard power management, battery charging, and expandable firmware capabilities.

</div>

---

# Overview

CB-01 is a custom electronic badge designed as a blend of PCB art, embedded systems, and hardware engineering.

The project combines functional embedded hardware with a cyberpunk-inspired aesthetic, resulting in a badge that serves both as a development platform and a wearable electronics project.

The board was designed in KiCad and demonstrates advanced PCB design practices, power management, component integration, and manufacturable hardware design.

---

# Features

- RP2040 Microcontroller
- USB Type-C Connectivity
- LiPo Battery Support
- MCP73831 Battery Charging Circuit
- Onboard 3.3V Regulation
- RGB LED Integration
- OLED Display Support
- External QSPI Flash Memory
- Power Switch
- Compact Wearable Form Factor
- Cyberpunk PCB Artwork
